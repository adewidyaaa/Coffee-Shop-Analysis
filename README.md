# Coffee Shop Analysis

## Repository Outline
1. README.md - Penjelasan gambaran umum project Coffee Shop Analysis
2. coffee_shop_analysis.ipynb - File notebook python yang berisikan pengolahan data kedai kopi
3. coffee_shop_dataset.csv - File csv berisikan data kedai kopi yang akan dianalisa


## Problem Background
Kedai kopi ingin menganalisa perilaku pelanggan, menu terlaris, peak hour, tren revenue dan fluktuasi penjualan yang terjadi selama 1 tahun terakhir. Tujuannya adalah untuk membantu dalam pengambilan keputusan bisnis dan meningkatkan strategi maupun revenue dan profit dari kedai kopi.

## Project Output
Project ini menghasilkan :
- Visualisasi berupa pie chart, horizontal bar chart, line chart dan bar chart
- Analisa statistik deskriptif dan statistik inferential
- Insight yang terkait dengan problem background kedai kopi
- Analisa dan kesimpulan dari semua perhitungan yang dilakukan

## Data
Sumber : Coffee Sales Dataset (Kaggle)  
Karakteristik Dataset :
- Jumlah kolom = 11 kolom. Setelah proses preprocessing kolom tersisa sebanyak = 8 kolom
- Jumlah baris = 3547 baris
- Tidak terdapat missing value

## Method
- Visualisasi : Pie chart, Bar chart dan line chart untuk menganalisa menu kopi terlaris, omset kopi tertinggi, peak hour dan hari dengan penjualan tertinggi
- Statistik Deskriptif : Mean, Median dan Mode untuk menganalisa rata-rata jumlah transaksi harian tiap bulan dalam 1 tahun terakhir
- Statistik Inferential : Metode ANOVA untuk mengetahui perbedaan revenue yang terjadi setiap bulan
